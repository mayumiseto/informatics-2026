# 第09回 retry フォローアップ

## 09_retryとは

`09_retry` は、第09回のヒント入り再挑戦用フォルダです。

元の `09/` フォルダを上書きしないために、別フォルダとして用意しています。

```text
09/        元の作業場所
09_retry/  ヒント入りの再挑戦用
```

## 触るファイルと実行するファイル

基本は次の通りです。

```text
functions.py を編集する
analysis.py を実行する
```

例：

```python
!python 09_retry/practice/1/analysis.py
```

この命令は、`analysis.py` というPythonファイルを実行しています。

## `functions.py` に書くもの

`functions.py` には、処理の部品となる関数を書きます。

例：

- 摂氏温度を華氏温度に変換する
- 指定した月だけを取り出す
- 平均値・最大値・最小値を計算する
- CSVファイルとして保存する

## `analysis.py` に書いてあるもの

`analysis.py` には、処理全体の流れが書いてあります。

例：

1. データを読み込む
2. `functions.py` の関数を呼び出す
3. 結果を保存する
4. 画面に表示する

## 大事なこと

このretryの目的は、完璧なコードを暗記することではありません。

```text
どのファイルを編集するか
どのファイルを実行するか
関数を直すと実行結果が変わること
```
